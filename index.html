Etiquetas

Para construir el código en HTML utilizaremos diversas etiquetas, las cuales nos ayudarán a definir los elementos dentro de nuesta págnia web.

Dentro de estas equitquetas podemos encontrar diversos elementos, algunos de los eleméntos badicos que componen una etiqueta es el "Inicio", "Contenido" y el "Cierre", esto se utiliza por ejemplo para escribir los títulos [(h1-h6)CONTENIDO(/h1-h6)], los párrafos [(p)CONTENIDO(/p)], etc. También tenemos los atributos, estos van dentro de las etiquetas, por ejemplo para escribir un URL. [(a herf = "URL" ) NOMBRE (/a)]

Se recomienda escribir en bajas y tener ojo con los espacios.

Algunas etiquetas pueden ir dentro de otras, como es el caso de las listas de definición, las listas ordenadas y las listas sin orden.

Listado de definición
Para formar esta lista se requiere de tres niveles, primero (dl)(/dl) el cual abre y cierra la lista de definiciones, en segundo lugar está (dt)(/dt) para colocar la definición del término, y por último está (dd)(/dd) utilizado para la descripción de la definición
Listado desordenado
Para este listado de utilizan dos niveles, dos etiquetas distintas y una dentro de la otra, (ul)(/ul) de "unorden list" para abrir y cerrar la lista y (li)(/li) para colocar los items de la lista
  • A
  • B
Listado Ordenado
Para este listado tambien se utilizan dos etiquetas y una dentro de la otra, estas don (ol)(/ol) de "orden list" para abrir y cerrar la lista y (li)(/li) para colocar los items de la lista
  1. A
  2. B
